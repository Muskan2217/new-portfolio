What I'm good at

Creating Responsive Layouts

Responsive web design makes your web page look good on all devices. Web pages can be viewed using many different devices:Desktops,Tablets & Phones. It is created by using HTML & CSS.

Reduces Repetition of Code

Stylesheets are getting larger,more complex,and harder to maintain. This is where a CSS pre-processor i.e SASS can help. SASS reduces repetition of CSS and therefore saves time.

Developing Client-side Validation

Before submitting data to the server,it is important to ensure all required form controls are filled out,in the correct format. This is called client-side form validation. It is developed by using Javascript & JQuery.
